# Complete Public Project Package

This repository is the complete public-safe GitHub package for the project.

Included:
- Full redacted technical report text.
- All extractable report media files (16 files).
- Electrical drawing package: PNG/SVG/DXF/DWG files (102 files).

Not included:
- Raw original submission archives or binary reports that may expose student IDs, emails, or private document metadata.
- Vendor/course reference material that is not part of the project implementation.
