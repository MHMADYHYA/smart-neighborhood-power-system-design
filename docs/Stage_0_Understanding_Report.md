# Stage 0 — דוח הבנה וסיווג מקורות (סופי)
**תאריך:** 11/06/2026 · בוצע לפני תחילת העריכה, כנדרש.

## מהות הפרויקט
תיק תכנון סופי (אבן דרך רביעית — דוח מסכם) למערכת חשמל במתח נמוך לשכונת מגורים חכמה, מסווג **אפשרות ב׳** לפי הנחיות רשם המהנדסים: 3 בניינים (A/B/C) × 15 קומות (קרקע טכנית + 14 מגורים × 4 דירות) = **168 דירות** בחיבור 3×25A; הזנה 22kV דרך **2×1000kVA** (יבשים, Dyn11, uk≈6%) אל **MDB דו-מקטעי 2500A / Icw 65kA** (2×ACB 1600A + Bus-tie); **PV 200kWp**, **BESS 250kW/500kWh**, **30×11kW EV + DLM**, **גנרטור 300kVA + ATS**, **UPS 20kVA/8.75kWh**, **קבלים 200kvar detuned**, **EMS/BMS**, הארקת **TN-S**, חלוקה תת-קרקעית. ביקוש: 1,312.7 / 1,603.7kVA → העמסה 65.6% / 80.2%.

## סתירות וערכים ישנים שאותרו (ותוקנו)
1. **זהות** — הקובץ נשא ניסוחי "דוח ביניים – אבן דרך שלישית" בשער, בתקציר, בפרק 2, בפרק 20 ובעשרות פסקאות ("ראשוני", "יושלם בדוח הסופי").
2. **UPS הפוך** — במסמך 8.75kVA/20kWh; הנתון המחייב: **20kVA/8.75kWh** → חושבו מחדש מזין (MCB 40A, 4×10), Ik (3.4kA) וזמן גיבוי (≈1.1h).
3. **סתירה פנימית בפרק 9** — חישוב 9.1 השתמש ב-490A/MCCB 630/"2×5×240" מול 574A/800A/2×(4×300) בשאר הדוח.
4. **"4×35" שרד** בטבלת מפל המתח (רייזר) בסתירה ל-9.7 → מודל הרייזר תוקן ו-ΔU מצטבר עודכן 4.0%→3.5%.
5. **Icu של ACB** — 50kA בחלק מהטבלאות מול 65kA באחרות → אוחד ל-65/65kA.
6. **לוח קומה** — 125A/10kA/80A בלתי עקביים → אוחד ל-MCCB 80A, Icu 25kA.
7. **Ik ללוח ציבורי/UPS** (11/10kA) לא תאם לכבל 4×10 → חושב מחדש 3.4kA.
8. **שרטוטים ישנים** הציגו UPS 8.75kVA, PUBLIC 100A, 50kA → הופקו מחדש.
9. **חוסרים מול דרישות הדוח המסכם** — אין פרק חלופות, אין BOQ סופי, נספחים חלקיים (חסרים C/I/J/M), אין טבלאות התאמה מלאות, אין Ik,min לכל לוח, תהודת פסים "תושלם" — הכל הושלם.

## איך נבדקה ההתאמה לחוק החשמל ותקנות החשמל
לכל נושא תכן הוצגה שרשרת ההוכחה המעשית (Ib → In/Ir → Iz′ → I2≤1.45Iz′ → ΔU → Ik max/min → Icu/Ics → שיטת הגנה/הארקה → החלטה), בתוספת: טבלת התאמה לפי 16 נושאי תכנון (פרק 2), סעיף ייעודי בכל פרק הנדסי, ושתי טבלאות בקרה בפרק 22. בהיעדר ציטוט נוסח מחייב — הניסוח "בהתאם לעקרונות חוק החשמל ותקנות החשמל בנושא..." בלבד, ללא המצאת מספרי סעיפים.

## החלטת ביצוע
התוכן ההנדסי בבסיס נמצא איכותי ועקבי ברובו; הביצוע התמקד ב: (א) הסבת הזהות לדוח מסכם; (ב) השלמת הפרקים/נספחים/טבלאות הנדרשים; (ג) תיקון הסתירות שלעיל; (ד) הפקת שרטוטים סופיים תואמי-ערכים והטמעתם; (ה) בקרת איכות ממוכנת. כל המספרים הקבועים של הפרויקט נשמרו ללא שינוי.
