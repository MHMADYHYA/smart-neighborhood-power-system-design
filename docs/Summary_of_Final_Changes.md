## 4. תוספות תוכן נדרשות
- **טבלת מצבי פעולה EMS** (7 מצבים × רשת/PV/BESS/גנרטור/UPS/EV) + **פסקת Interlock גנרטור–PV/BESS** (איסור הקבלה, חציצת ATS, anti-islanding, anti-parallel ב-EMS).
- טבלת עומסי חירום סופית עם עמודות "חייב גנרטור / חייב UPS / זמן גיבוי / התאמה".
- טבלת בחירת כבלים סופית — 14 עמודות × 18 מסלולים (כולל קבלים, תאורה, BMS, UPS→קריטי).
- טבלת מפלי מתח סופית (R/X/cosφ/מצטבר/גבול) וטבלת זרמי קצר סופית (Ik max/min, Icu/Ics).
- טבלת כיול הגנות וסלקטיביות (LSIG מלא) + בדיקת סלקטיביות לכל השרשראות (ראשית/EV/BESS/PV/גנרטור/UPS).
- טבלת הארקה והשוואת פוטנציאלים עם חתכי PE ובדיקות נדרשות לכל רכיב; טבלת פסי צבירה לכל הלוחות; טבלת תאורה סופית; פסקת בטיחות BESS; חלופת תאורת חוץ.
- בוטלו צילומי דפי קטלוג (3 איורים) לפי איסור ההנחיות; המקורות נשארו בטבלת מקורות ובביבליוגרפיה; נוספו הנחיות 23/02/2026 לרשימת המקורות.

## 6. איך נבדקה ההתאמה לחוק החשמל
לכל נושא הוצגה שרשרת התכן המעשית: זרם עומס ← מבטח (In/Ir) ← כושר נשיאה מתוקן (Iz′, מקדמי C1×C4) ← I2≤1.45×Iz′ ← מפל מתח ← זרם קצר (מקס׳+מינ׳) ← Icu/Ics ← שיטת הגנה/הארקה (TN-S, Zs, RCD) ← החלטת התאמה. היכן שנוסח הסעיף אינו מצוטט — נכתב "בהתאם לעקרונות חוק החשמל ותקנות החשמל בנושא..." ללא המצאת מספרי סעיפים.
