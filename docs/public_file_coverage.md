# Public File Coverage

This repository includes every project file that is useful and safe for public GitHub review.

Added/covered:
- Rendered final report pages from page 2 onward (156 images).

Intentionally excluded from the public repo:
- Raw DOCX/PDF/ZIP/RAR/SLS/PPT files with possible private metadata.
- Course lecture/reference PDFs that are not authored project deliverables.
- Generated caches, pycache files, temporary lock files, and duplicate archives.

Reason: raw academic submissions and course materials can contain student IDs, emails, document metadata, or third-party lecture/reference content. Public repositories keep the technical work, source, media, reports, drawings, and results without exposing private material.
